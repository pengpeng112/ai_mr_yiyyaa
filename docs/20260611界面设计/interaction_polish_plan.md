# 前端交互细节打磨方案

> 目标：在不改变业务逻辑的前提下，全面提升系统的交互质感和用户体验。
> 技术栈：Vue3 + Element Plus + 纯 CSS/JS，无需引入新依赖。

---

## P1 详细方案（本轮实施）

### 1. 数字滚动动画（CountUp）

**效果**：KPI 数字从 0 平滑滚动到目标值，持续 800ms。

**实现**：
- 新建 `static/scripts/utils/count-up.js` 工具函数
- 在 Dashboard KPI 卡片、各页面指标细条中使用
- 使用 `requestAnimationFrame` 实现，无依赖

**应用位置**：
- Dashboard：今日推送总数、成功率、不一致数、高危数、待处理反馈等
- 患者质控：患者总数
- 质控反馈：近期不一致、高风险、待确认、已关闭
- 推送日志：总记录、跳过、失败、未复核
- 前置机告警：告警总数、发送失败、未查看

### 2. 全局空状态组件

**效果**：所有列表无数据时，显示统一的插画 + 引导文案 + 操作按钮。

**实现**：
- 新建 `static/templates/common/empty-state.html` 片段
- 使用 SVG 插画（内联，无需图片文件）
- 不同场景不同插画：无数据、无搜索结果、无权限、错误

**空状态类型**：
| 类型 | 插画 | 文案 | 操作 |
|------|------|------|------|
| 无数据 | 空盒子 | "暂无XX记录" | 新建按钮 |
| 无搜索结果 | 放大镜 | "未查询到符合条件的记录" | 清除筛选 |
| 无权限 | 锁图标 | "您没有权限查看此内容" | 联系管理员 |
| 加载失败 | 警告 | "加载失败，请重试" | 重新加载 |

### 3. 暗色模式

**效果**：点击顶部栏切换按钮，整个系统切换为暗色主题。

**实现**：
- 在 `:root` 定义亮色 CSS 变量
- 在 `:root[data-theme="dark"]` 定义暗色覆盖
- 顶部栏增加月亮/太阳图标切换按钮
- 使用 `localStorage` 记住用户偏好

**颜色映射**：
| 亮色 | 暗色 |
|------|------|
| `--surface-page: #f3f6fa` | `#0f172a` |
| `--surface-card: #ffffff` | `#1e293b` |
| `--text-primary: #262626` | `#e2e8f0` |
| `--border-subtle: #e5e7eb` | `#334155` |

### 4. 页面切换过渡动画

**效果**：切换菜单时，内容区域有淡入 + 轻微上移动画。

**实现**：
- 在 `index.html` 的 `<main class="content">` 内层包 `<transition name="page-fade">`
- CSS 定义 `.page-fade-enter-active` 过渡

### 5. 按钮操作反馈增强

**效果**：
- 主按钮点击时有轻微缩放动画
- 成功操作后按钮短暂变绿打勾
- 危险按钮 hover 时有红色波纹

**实现**：
- CSS `:active` 伪类 + transform
- 操作成功后临时设置 `justSucceeded` 状态

## P2 详细方案（后续轮次）

### 6. 表格交互增强
- 行 hover 时显示操作按钮（默认隐藏）
- 双击行直接打开详情
- 右键菜单（复制、详情、重推）

### 7. 全局快捷键
- `Ctrl+K`：打开全局搜索
- `Ctrl+R`：刷新当前页
- `Esc`：关闭弹窗
- `←/→`：详情弹窗翻页（已有部分）

### 8. 卡片悬浮微动效
- KPI 卡片 hover 时轻微上浮 + 阴影增强
- 图表卡片 hover 时边框高亮

### 9. 加载骨架屏扩展
- 将 Dashboard 骨架屏模式扩展到所有页面
- 表格加载时显示骨架行

### 10. 消息提示优化
- 成功消息从右上角滑入，带绿色打勾图标
- 错误消息持续 5 秒，可手动关闭
- 批量操作进度条消息
